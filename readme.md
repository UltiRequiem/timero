# Timero

> https://timero.ultirequiem.com

A website to see what time is in any timezone.

## Characteristic

- Responsive Design

- [Packup](https://packup.deno.dev) as Bundler

- TypeScript

- CI and CD

  Deno/Browser/Node.js
