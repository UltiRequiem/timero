# Timero

Webapp to see what time is in any timezone.

## Deploy

https://ultirequiem.github.io/timero

## Characteristic

- Responsive Design

- Uses [Packup](https://packup.deno.dev) as Bundler

- Written in TypeScript

- CI and CD
